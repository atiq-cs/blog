Notepad++

notepad++ flavor uses \1, \2 and so on to refer to matched terms under ().

How to replace NSLog(@x) with printf(x) In find text box we put,

(NSLog\(@)(.*) (;)

In replace text box we put,

printf(\2

Copying from github then removing line numbers in front of all lines removing them using notepad++ regular expression,

find        : ^([0-9][0-9][0-9] )(.*)
replacement : \2

regular expression for clone repo, dependency repos given list of repo names,

clone_repo /builds/atiq/jv8/\1 \1 false

Remove beginning line numbers from the output of history command, ^ 7..

Remove all numbers from bash history files,

find        : ^\s*(\d|\d\d|\d\d\d)\s\s(\S)
replacement : \2
